Robopoly Control Center

This application was made to show off the Robopoly robotics platform capabilities. It allows to fetch the robot sensors values in real time and display them in various ways. The user can control the mobile platform or let it perform tasks such as line following and obstacle detection.

To use any of the activities start by loading the Arduino program on the PRismino and then connect to the device by using the upper menu.

The mobile platform works best when used with the official Robopoly 3D-printed base.
